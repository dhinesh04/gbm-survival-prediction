# Multi-omics Data Integration using Graph Convolutional Networks for Glioblastoma Survival Prediction

## How to run?

### Clone repository

git clone https://github.com/dhinesh04/gbm-survival-prediction.git

### Creating/Activating virtual environment

python3.9 -m venv env
source env/bin/activate

### Installing requirements

pip install -r requirements.txt

### Downloading raw data

Download the data used in this study from https://doi.org/10.6084/m9.figshare.31807621 and store it in a directory called "cbioportal_data"

### Run the file

bash run.sh

The outputs will be stored in logs directory for easy investigation.
